# Dice Roll Generator 🎲

A responsive, mobile-first Dice Roll Generator built with vanilla JavaScript and modern CSS. Features include dynamic DOM manipulation, input validation, and fluid typography using CSS Clamp

## 🚀 Features
- **Dynamic Rendering:** Dice faces are generated instantly based on user input.
- **Mobile-First Design:** Fully responsive layout using `clamp()` and `dvh` units.
- **Input Validation:** Prevents invalid entries (0 or >100) to ensure app stability.
- **Accessibility:** Semantic HTML and ARIA-friendly image alt tags.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox (with wrapping logic), Custom Properties, and Responsive Design.
- **JavaScript:** DOM Manipulation and Array methods.
